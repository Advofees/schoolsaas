# Teacher

## permissions
- view students
- update student attendance
- view student attendance
- view student grades
- updated student grades

## Features
- manage classroom

    - logged in teacher
        - students view:
            - all current year students
            - can click a student to view detailed info
            - can click a student to edit
        - classroom attendance view
            - can add attendance
            - can edit attendance
            - sortable by:
                - day
                - month
                - week
                - student name
                - PRESENT | ABSENT

        - classrom grades view
            - sortable by:
                - academic term
                - exam
                - grades min
